# dio\_excel\_dash\_vendas\_xbox



Projeto\_001, projeto de criação de um dashboard de vendas com planilha base de games para xbox.
OBS\* A ideia do gráfico é mais nas funcionalidades e não na parte visual.



-- Esse projeto é a versão 1.0 do exercício de módulo das aulas de Copilot do Felipão.

Versão atual contém:

Base de dados,

Cálculos e guia do DashBoard.



Foi feito algumas modificações usando o Microsoft Copilot conforme ensinado nas aulas.



Aqui serão descritos os cálculos e alterações:



Pergunta 01: Qual o faturamento Total de vendas de planos anuais (contendo todas as assinaturas agregadas)

Pergunta 02: Qual o faturamento Total de vendas de planos anuais, separado por auto renovação.

Pergunta 03: Qual o valor total de cupons aplicados por tipo de assinatura e planos?

Pergunta 04: Qual o Ticket Médio por assinante em cada tipo de assinatura?



-- Tabelas Dinâmicas para cada uma das questões com Filtros.

-- Card de resultados dinâmicos vinculados;

-- Filtro vinculado.





Valores: Valor Total em Vendas - Valor Médio por Plano - Valor Médio por Assinatura em Período - Preço Médio por Assinatura.
